# kobo-book-exporter

Export Kobo Book Highlight (.md) & Book List (.csv, .json & .md).

[Kobo Book Exporter Webapp](https://mrstruijk.github.io/kobo-book-exporter)
![Screenshot](https://mrstruijk.github.io/kobo-book-exporter/img/screenshot.png)


## (Install)

```shell
yarn install
```

## (Usage)

(Create files)
```shell
yarn run build
```

(Run)
```shell
yarn run serve
```
